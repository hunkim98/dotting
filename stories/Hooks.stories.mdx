# Using hooks for interaction!

#### How to setup hooks

To manipulate the Dotting component you should have a ref object attached to the Dotting Component.
You can use React's `useRef` to get the DottingRef refobject of the Dotting Component.
After the ref object is set, you can use the `useDotting` or `useBrush` hook and pass the ref as its parameter.

Below is the example setup for using the `useDotting` hook.
The example below shows how you can clear the canvas.

#### Setup Example (Clearing the Canvas)

```ts
export const Component = () => {
  const ref = useRef<DottingRef>(null);
  const { clear } = useDotting(ref);
  return (
    <div>
      <Dotting ref={ref} width={"100%"} height={300} />
      <button onClick={clear}>Clear Canvas</button>
    </div>
  );
};
```

#### Styling

You can add styles to Dotting component via `React.CSSProperties` <br/>
`gridStroke` props to control grid's appearances <br/>
`background` props to control base layer's appearances <br/>

```ts
export const Component = () => {
  return (
    <div>
      <Dotting
        gridStrokeColor={gridStrokeColor}
        gridStrokeWidth={gridStrokeWidth}
        style={{ borderColor: "red" }}
        backgroundColor="#FFD4D2"
      />
    </div>
  );
};
```
